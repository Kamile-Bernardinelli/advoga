# Story 1.5 — Dashboard + Countdown

> **ID:** 1.5 · **Epic:** EP-1 Combat Core
> **Status:** Draft
> **Agente executor:** Dex (@dev) — UX spec de Uma (@ux-design-expert) como input
> **Dependência:** Story 1.3 concluída (dados de `diagnostico` disponíveis)
> **DoD Drop 1 que cobre:** "ver quantos dias faltam" + "acompanhar minha evolução" — componente de orientação temporal e progresso geral

---

## User Story

**Como** Kamile,
**quero** ter uma dashboard principal que mostre imediatamente quantos dias faltam para a prova, como estou evoluindo por matéria ao longo do tempo e um resumo do meu estado atual,
**para que** eu sinta o peso do prazo e saiba de uma relance se estou avançando ou estagnando.

---

## Contexto Técnico

- Subsistema 7 do brief: Dashboard & Evolução, desktop-first.
- Data fixa: **06/09/2026**. Countdown calculado em dias inteiros.
- Séries temporais: taxa de acerto por matéria ao longo das sessões (eixo X = data da sessão, eixo Y = taxa).
- Desktop-first (D-04): layout otimizado ≥ 1280px. Elementos visíveis sem scroll na fold principal.
- O countdown é o elemento de maior destaque visual ("de extrema importância" — verbatim Kamile, RF-10).

---

## Acceptance Criteria

- [ ] AC-1.5.1 (RF-10): Countdown até 06/09/2026 exibido em dias na homepage do dashboard. Atualizado diariamente. Exemplo: "**77 dias** para a prova".
- [ ] AC-1.5.2 (RF-09): Gráfico de séries temporais: taxa de acerto geral (todas as questões) ao longo das sessões. Linha de tendência visível.
- [ ] AC-1.5.3 (RF-09): Gráfico de barras de estado atual: taxa de acerto por matéria, ordenado por fraqueza (piores primeiro). Consistente com diagnóstico da Story 1.4.
- [ ] AC-1.5.4 (RF-11): Card "Última sessão": data, prova feita, resultado (acertos/erros). Link para o relatório completo.
- [ ] AC-1.5.5 (RF-09): Card "Total histórico": total de questões feitas, total de sessões, taxa geral.
- [ ] AC-1.5.6 (RF-09): Indicador visual de tendência geral (melhora / estabilidade / piora) baseado nas últimas 3 sessões.
- [ ] AC-1.5.7 (D-04): Layout desktop-first: countdown + gráfico principal + 3 cards acima da fold em tela de 1280px.
- [ ] AC-1.5.8: Responsivo mínimo para celular (consulta rápida — não otimizado, mas não quebra).
